Accessibility News

New Accessibility Features in iOS 15

Apple has announced new accessibility features in the latest version of its mobile operating system, iOS 15. The new features include improved voice recognition for people with speech impairments, a new feature called "Focus" that allows users to customize notifications and reduce distractions, and enhancements to the "Magnifier" tool that makes it easier for users with low vision to see text and images on their device.

Microsoft Launches AI-Powered Captioning Tool

Microsoft has launched a new AI-powered captioning tool that uses speech recognition technology to automatically generate captions for videos. The tool, called "Live Captions," is available on the latest versions of Windows and Android, and is designed to make videos more accessible to people with hearing impairments.

Google Releases Accessibility Toolkit for Developers

Google has released a new accessibility toolkit for developers, called "Accessibility Scanner," that provides automated suggestions for improving the accessibility of Android apps. The toolkit includes a set of tools and resources that developers can use to test their apps for accessibility issues and make fixes to improve their accessibility.
